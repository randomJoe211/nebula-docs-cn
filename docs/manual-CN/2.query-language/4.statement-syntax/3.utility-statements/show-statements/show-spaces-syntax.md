# SHOW SPACES 语法

```ngql
SHOW SPACES
```

`SHOW SPACES` 列出 **Nebula Graph** 集群中的所有图空间。

例如：

```ngql
nebula> SHOW SPACES;
====================
| Name             |
====================
| basketballplayer |
--------------------
```

参考[这里](../../1.data-definition-statements/create-space-syntax.md)创建图空间。
